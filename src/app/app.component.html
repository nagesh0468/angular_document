Learning Angular

Basic Setup for Angular

1. Install Node.js

First, install Node.js from the official website.

2. Install Angular CLI

After installing Node.js, install Angular CLI globally using the command:

npm install -g @angular/cli

3. Verify Installation

Check if Angular CLI is installed correctly by running:

ng version

If the installation was successful, it will display the Angular CLI version.

4. Create a New Angular Project

Run the following command to create a new Angular app:

ng new app-name

Replace app-name with your preferred project name.

5. Run the Angular App

Navigate to your project folder and start the development server:

cd app-name
ng serve

This will start your application, and you can access it in your browser at http://localhost:4200/.

6. Need Help?

For more details, check out the official Angular guide.

If you face any issues, feel free to ask for help.

Happy Coding! 🚀

Learning Angular

Components and Data Binding

In an Angular project, your main code is inside the app folder. This folder contains HTML, CSS, and TypeScript files for your components.

To display a simple message, open the HTML file of your component (e.g., app.component.html) and add this line:

<h1>Welcome to Angular</h1>

Now, run your Angular app using:

ng serve

Open your browser and go to http://localhost:4200/. You should see "Welcome to Angular" displayed on the page!

I think you are so happy to see this welcome 😁!

That's how simple it is to work with Angular components and templates.

Learning Angular

Property, Class, and Style Binding in Angular

Now that you've learned about text interpolation, let's explore binding attributes dynamically using:

  • Property Binding – for setting element attributes dynamically
  • Class Binding – for dynamically applying CSS classes
  • Style Binding – for dynamically setting styles

### 1️⃣ Setup in TypeScript File (app.component.ts)

export class AppComponent {
    message = 'Welcome to Angular!';
    imagePath = 'assets/angular-logo.png'; // Property binding
    styleClass = 'highlight'; // Class binding
    textColor = 'blue'; // Style binding (dynamic text color)
    bgColor = 'lightgray'; // Style binding (background color)
    currentDate = new Date().toDateString(); // Display current date
  }

### 2️⃣ Update the HTML File (app.component.html)

Now, let's dynamically bind properties, classes, and styles:


<h1>{{ message }}</h1>

<!-- Property Binding for Image -->
<img [src]="imagePath" alt="Angular Logo">

<!-- Class Binding -->
<p [class]="styleClass">This text has dynamic styling!</p>

<!-- Style Binding -->
<p [style.color]="textColor" [style.backgroundColor]="bgColor">
  This text has a dynamically applied color and background!
</p>

<!-- Date Binding -->
<p>Today's Date: {{ currentDate }}</p>

### 3️⃣ Run Your Angular App

ng serve

Now, open http://localhost:4200/ in your browser.

🎉 Browser Output

Here’s what you will see in the browser:

Welcome to Angular!

This text has dynamic styling!

This text has a dynamically applied color and background!

Today's Date: Thu Mar 06 2025

WOW! 🎉 You did it! The dynamic data is working perfectly! 😃

Now, you’ve successfully used:

  • Property Binding to display the image dynamically
  • Class Binding to apply styling dynamically
  • Style Binding to change text and background color dynamically
  • Text Interpolation to display the current date dynamically

Event Binding in Angular

Now that you've learned about property, class, and style binding, let's explore Event Binding in Angular.

What is Event Binding?

Event binding allows us to handle user actions (like clicks, key presses, mouse movements) by binding them to functions in our TypeScript file.

### 1️⃣ Setup in TypeScript File (app.component.ts)

export class AppComponent {
    message = 'Click the button to see the magic! 🚀';

    changeMessage() {
      this.message = 'Wow! You just clicked the button! 🎉';
    }
  }

### 2️⃣ Update the HTML File (app.component.html)

Now, let's use event binding to trigger the function when the button is clicked:


<h1>{{ message }}</h1>

<!-- Event Binding for Button Click -->
<button (click)="changeMessage()">Click Me!</button>

### 3️⃣ Run Your Angular App

ng serve

Now, open http://localhost:4200/ in your browser.

🎉 Browser Output

Before Clicking the Button:

Click the button to see the magic! 🚀

After Clicking the Button: (The text updates dynamically!)

Wow! You just clicked the button! 🎉

OH WOW! 🎉 The text changed instantly! That’s event binding in action! 😃

Now, you’ve successfully used:

  • Event Binding to capture user interaction
  • Function Execution to update the UI dynamically
  • Text Interpolation to reflect changes in real-time
